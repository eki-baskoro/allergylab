# AllergyLab

[![AllergyLab CI/CD](https://github.com/eki-baskoro/allergylab/actions/workflows/docker.yml/badge.svg)](https://github.com/eki-baskoro/allergylab/actions/workflows/docker.yml)

AllergyLab is a comprehensive application designed to help users manage and track allergies, food sensitivities, and related health information.

## Features

- **Allergy Tracking**: Log and monitor allergies and reactions
- **Food Sensitivity Management**: Track food intolerances and sensitivities
- **Reaction Logging**: Document symptoms and severity levels
- **Health History**: Maintain a comprehensive record of allergic reactions
- **Search & Filter**: Easily find allergy information
- **Data Export**: Export your allergy records for medical professionals

## Installation

Clone the repository:
```bash
git clone https://github.com/eki-baskoro/allergylab.git
cd allergylab
```

Install dependencies:
```bash
npm install
```

## Usage

Start the application:
```bash
npm start
```
